Create backend/app/api/dashboard.py with APIRouter(prefix="/api/dashboard", tags=["dashboard"]):

**GET /api/dashboard/stats** (per D-01, D-05, DASH-01):
- Depends on get_current_user from app.api.deps.
- The current_user object already has plan, credits_remaining, subscription_expires_at from the User model.
- Query renders table: COUNT(*) WHERE user_id = current_user.id for renders_total.
- Query renders table: COUNT(*) WHERE user_id = current_user.id AND created_at >= first_day_of_current_month for renders_this_month. Use `datetime.now().replace(day=1, hour=0, minute=0, second=0, microsecond=0)` for month start.
- Derive credits_total from plan: free=3, starter=50, business=999 (unlimited represented as 999). Use a simple dict mapping: `{"free": 3, "starter": 50, "business": 999}`.
- Return DashboardStatsResponse with all fields populated.

Create backend/app/api/renders.py with APIRouter(prefix="/api/renders", tags=["renders"]):

**GET /api/renders** (per D-02, DASH-02, DASH-04):
- Depends on get_current_user.
- Query params: limit (int, default=20, max=100), offset (int, default=0).
- Query renders WHERE user_id = current_user.id ORDER BY created_at DESC, with LIMIT and OFFSET.
- For each render, generate a presigned GET URL from MinIO if output_url is set (the output_url field stores the MinIO object path like "rendered/{user_id}/{render_id}.png"). Use `minio_client.presigned_get_object("rendered", render.output_url, expires=timedelta(hours=1))`. Replace render.output_url with the presigned URL in the response.
- Also query total count: SELECT COUNT(*) FROM renders WHERE user_id = current_user.id.
- Return RenderListResponse with renders list, total, limit, offset.

**GET /api/renders/{render_id}/download** (per DASH-03):
- Depends on get_current_user.
- Query render by id. If not found or user_id mismatch: 404.
- If output_url is None (render not complete): 400 {"error": "Render not ready"}.
- Generate presigned GET URL from MinIO with 1-hour expiry.
- Return 200 {"download_url": presigned_url, "filename": "{marketplace}_{render_id}.png"}.

**DELETE /api/renders/{render_id}** (per D-03, DASH-03):
- Depends on get_current_user.
- Query render by id. If not found or user_id mismatch: 404.
- Delete the rendered file from MinIO: minio_client.remove_object("rendered", render.output_url). Wrap in try/except (file might already be gone).
- Delete the render record from DB.
- Return 204 No Content.

For the MinIO client: create a helper function or import from services/minio.py. Check if backend/app/services/minio.py has a get_minio_client() or similar -- if so, reuse it. If not, create a local helper using `minio.Minio(settings.s3_endpoint.replace("http://", ""), settings.s3_access_key, settings.s3_secret_key, secure=False)`. Remove the http:// prefix since Minio() expects host:port without scheme.
